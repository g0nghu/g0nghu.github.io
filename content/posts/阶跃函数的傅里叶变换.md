## 阶跃函数$\theta(t)$的傅里叶变换

阶跃函数
$$\theta(t) = \left\{
\begin{aligned}
1, \ & t \geq 0 \\
0, \ & t < 0
\end{aligned}\right.$$

此外我们定义
$$f_a(t) = \left\{
\begin{aligned}
e^{-at}, \ & t \geq 0 \\
0, \  & t < 0
\end{aligned}\right.$$

可以看出
$$\lim_{a\rightarrow0^+}f_a(t) = \theta(t)$$

对$f_a(t)$的F.T.是
$$\mathcal{F}[f_a(t)] = \int_0^\infty e^{-at-i\omega t} \mathrm{d}t = \frac{1}{a+i\omega}$$
对它取极限得到$\theta(t)$的F.T.
$$\mathcal{F}[\theta(t)] = \lim_{a\rightarrow0^+}\mathcal{F[f_a(t)]} = \lim_{a\rightarrow0^+}\frac{1}{a+i\omega}$$
进一步，将之展开成虚部和实部
$$\mathcal{F}[\theta(t)]=\lim_{a\rightarrow0^+}\frac{a-i\omega}{a^2+\omega^2}$$
实部给出一个$Dirac$函数，要确定系数$C = \lim_{a\rightarrow 0^+}\int \frac{a\mathrm{d}\omega}{a^2+\omega^2} = \pi$，所以有
$$\mathcal{F}[\theta(t)] = \pi\delta(\omega)+\frac{1}{i\omega}$$

此外有两个有用的积分
$$\int_0^\infty e^{-i\omega t}\mathrm{d}t = \pi\delta(\omega)+\frac{1}{i\omega}$$
$$\int_0^\infty e^{-(i\omega+\beta)t}\mathrm{d}t = \frac{1}{\beta + i\omega}(\beta> 0)$$
